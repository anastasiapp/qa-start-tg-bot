# Инструкция по использованию бота

## 🎯 Реализованный функционал

### Для админа:

1. **Создание события:**
   - Команда `/newevent`
   - Формат: `Название | 2025-12-15 19:00 | 60 | https://meet.link | public`
   - После создания показывается карточка с кнопками управления

2. **Карточка события для админа:**
   - 📢 **Оповестить подписчиков** — отправляет уведомление всем подписанным пользователям
   - ✏️ **Редактировать** — редактирование (будет реализовано позже)
   - ❌ **Отменить** — отменяет событие и уведомляет подписчиков

### Для подписчиков:

1. **Просмотр события:**
   - Команда `/start` — показывает список ближайших публичных событий
   - Ссылка `/start event_<ID>` — открывает карточку события

2. **Карточка события для подписчика:**
   - 🔔 **Подписаться** / ✅ **Подписка активна** — подписка на уведомления
   - 📅 **Добавить в календарь** — генерирует ссылку Google Calendar
   - ❌ **Отписаться** — отписка от события

## 📋 Флоу работы

1. **Админ создает событие:**
   ```
   /newevent
   Sprint Review #5 | 2025-12-15 19:00 | 60 | https://meet.google.com/... | public
   ```
   → Получает карточку события с кнопками управления

2. **Админ оповещает подписчиков:**
   - Нажимает кнопку "📢 Оповестить подписчиков"
   - Все подписанные пользователи получают карточку события в личные сообщения

3. **Пользователь подписывается:**
   - Находит событие через `/start` или по ссылке
   - Нажимает "🔔 Подписаться"
   - Карточка обновляется, показывая статус подписки

4. **При отмене события:**
   - Админ нажимает "❌ Отменить"
   - Все подписчики автоматически получают уведомление об отмене

## 🔧 Настройка

В файле `.env` должны быть:
```env
BOT_TOKEN=your_telegram_bot_token
ADMINS=123456789,987654321
```

Чтобы узнать свой ID для добавления в ADMINS, используйте команду `/whoami`.

## 📝 Примечания

- События сохраняются в SQLite базе данных (`data.db`)
- Уведомления отправляются только в личные чаты (не в каналы)
- Оповещения отправляются только по кнопке (не автоматически)
- При отмене события все подписчики автоматически уведомляются
