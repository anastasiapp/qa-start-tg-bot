## ⚠️ Что нужно для запуска

1. Создать файл `.env` с переменными:
   ```env
   BOT_TOKEN=your_telegram_bot_token
   ADMINS=your_telegram_user_id
   ```

2. Запустить бота:
   ```bash
   npm run dev
   ```
